# career-ops-dashboard

Live dashboard for Marvis Osazuwa's career-ops pipeline.

- **URL:** https://marz1307.github.io/career-ops-dashboard/
- **Source data:** Notion Applications DB
- **Refresh cadence:** auto-rebuilt after every career-ops routine (~6-8 times/day)
- **Auto-publish:** `publish-dashboard.sh` in [marz1307/MY_career-ops](https://github.com/marz1307/MY_career-ops) pushes `index.html` here whenever the pipeline produces new state.

Not affiliated with or endorsed by any company listed in the data. Read-only snapshot of personal job-search activity.
